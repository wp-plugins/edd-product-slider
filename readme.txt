== Installation ==

* Install it as a regular WordPress plugin

* Active the plugin.

* Use this [wpb-edd-slider] shortcode, where you need to show the product slider slider.




== Frequently asked questions ==


= I use shortcode but it's not showing the slider? =
Make sure you put the shortcode in text mood.

= Slider not looking as it's swhown in demo? =
We test the plugin with all default theme, if your theme css conflict with the plugin style you may need to edit the plugin css. You can add your css code on main.css file.
